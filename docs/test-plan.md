## Affected Pages/Routes
- `/` (홈 대시보드) — 스트릭, 진행률, 오늘 할 미션 표시 확인
- `/explorer` (탐색기) — 카테고리 필터, 시즌 필터, 공개 업적 목록
- `/public/[id]` (공개 업적 상세) — 참여/나가기, 댓글, 참여자 목록
- `/my` (프로필) — 달성 업적 배지, 통계 표시
- `/missions/[id]` — 미션 완료/취소, 축하 애니메이션 트리거

## Key Interactions to Verify
- 공개 업적 참여 → 미션 자동 생성 on `/public/[id]`
- 미션 완료 체크 → 스트릭 업데이트 on `/` 홈
- 업적 달성 → 축하 화면(confetti) 트리거
- 카테고리별 업적 필터링 on `/explorer`
- 시즌 업적 마감 임박 표시 on `/explorer`
- 로컬 알림 예약/취소

## Edge Cases
- 스트릭: 어제 미션 완료 안 했을 때 리셋 확인
- 시즌 업적: 마감일 지난 업적 참여 시도 → 차단 or 표시
- 카테고리 없는 업적 필터링 시 표시 방식
- 공개 업적 참여 후 즉시 나가기
- 이미 참여 중인 업적에 재참여 시도

## Critical Paths
- E2E #1: 인증(가입→로그인→토큰 리프레시) 전체 플로우
- E2E #2: 공개 업적 참여 → 미션 생성 → 미션 완료 → 업적 달성 전체 플로우
- 서비스 테스트: PublicAchievementService, MissionService, AchievementService, AccountService 핵심 메서드
