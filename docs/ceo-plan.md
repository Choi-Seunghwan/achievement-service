## Vision

### 10x Check
현재: "10개 시드 업적이 있는 목표 관리 앱"
10x 버전: "나의 성장 여정을 시각화하고, 같은 목표를 가진 사람들과 연결해주는 성취 플랫폼"

- 업적 달성 시 "성장 카드" 자동 생성 → SNS 공유 최적화
- 시즌제 업적으로 긴급성 + FOMO
- 업적 추천 엔진 (유저 참여 기반)
- 프로필 = 성장 포트폴리오
- 리마인더로 "잊어버림" 문제 해결

### Platonic Ideal
앱을 열면 내 성장 대시보드. 연속 달성 일수, 진행 중인 업적 진행률, 오늘 할 미션.
아래로 스크롤하면 "당신에게 맞는 새로운 도전" 카드. 참여 → 매일 푸시 알림 → 미션 완료 시 햅틱+애니메이션 → 업적 달성 시 confetti+성장 카드 → 인스타그램 스토리 공유 → 친구가 설치 → 바이럴 루프.

사용자가 느끼는 감정: "나는 해내는 사람이고, 그걸 세상에 보여줄 수 있다."
